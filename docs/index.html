interspace/magic-theme

this is the @magic-themes webboot theme. It is being used as the foundational theme for all interspace pages.

currently, this theme is being used on the following pages:

  • interspace.chat

installation

npm install --save --save-exact interspace/magic-theme

usually, interspace pages also use the project theme

usage

require

in /config.mjs:

// /config.mjsexport default {  // ... other config  THEME: ['project', 'interspace/magic-theme'],}

theme vars

colors

this theme exports and uses the following colors.

  • vars.white: #ffffff
  • vars.neutral: #5a5a5a
  • background
    • vars.background.light: #f3f3f3
    • vars.background.dark: #434343
  • link
    • vars.link.light: #e29a00
    • vars.link.dark: #e29a00
    • link.hover
      • vars.link.hover.light: #dddddd
      • vars.link.hover.dark: #999999
  • primary
    • vars.primary.neutral: #e29a00
    • vars.primary.light: #434343
    • vars.primary.dark: #f3f3f3
  • text
    • vars.text.light: #434343
    • vars.text.dark: #f3f3f3

max page width

vars.maxWidth: 1000px

fade duration:

vars.fadeDuration should be used for all css animations

vars.fadeDuration: 500ms

breakpoints

the following width breakpoints exist

  • vars.widths.tablet: 500px
  • vars.widths.laptop: 1024px
  • vars.widths.desktop: 1600px
