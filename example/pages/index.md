# interspace/magic-theme

this is the [@magic-themes](https://github.com/magic-themes) webboot theme.
It is being used as the foundational theme for all interspace pages.

currently, this theme is being used on the following pages:

* [interspace.chat](https://interspace.chat)

### installation

`npm install --save --save-exact interspace/magic-theme`

usually, interspace pages also use the [project theme](https://github.com/magic-themes/project)

## usage

### #usage require

in /config.mjs:

```
// /config.mjs

export default {
  // ... other config
  THEME: ['project', 'interspace/magic-theme'],
}
```
